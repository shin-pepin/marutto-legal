## 紹介文（100文字以内）

特定商取引法の表記ページをかんたん3ステップで作成。入力・テンプレート選択・プレビューだけで法令準拠のページが完成します。

---

## 詳細説明（500文字以内）

「まるっと法務作成」は、日本のEC事業者が特定商取引法に基づく表記ページを簡単に作成できるShopifyアプリです。

ウィザード形式の3ステップで、必要な情報を入力するだけ。テンプレートを選んでプレビューを確認したら、ワンクリックでShopifyストアにページを作成できます。

主な特徴：
- ガイド付きフォームで入力漏れを防止
- 法令に準拠したテンプレートを用意
- リアルタイムプレビューで仕上がりを確認
- 作成後の編集・更新もかんたん
- ダッシュボードで公開状態を一目で確認

面倒な法務ページの作成を、もっとシンプルに。

---

## 機能リスト

1. 特定商取引法に基づく表記ページをウィザード形式で簡単作成
2. 法令準拠テンプレートで入力漏れ・記載ミスを防止
3. リアルタイムHTMLプレビューで公開前に仕上がりを確認
4. 作成済みページの編集・更新にワンクリックで対応
5. ダッシュボードでページの公開状態・最終更新を一覧管理

## スクリーンショット撮影チェックリスト

以下のスクリーンショットを撮影し、Shopify Partner Dashboardにアップロードする。

- [ ] **ダッシュボード画面**: ページ一覧が表示されている状態（公開済み・非公開のステータス表示）
- [ ] **ステップ1 — 情報入力**: フォームにサンプルデータが入力された状態
- [ ] **ステップ2 — テンプレート選択**: テンプレートが選択されている状態
- [ ] **ステップ3 — プレビュー**: 生成されたHTMLプレビューが表示されている状態
- [ ] **成功バナー**: ページ作成成功後のダッシュボード画面

### 撮影ガイド

- 解像度: 1600x900px以上
- ブラウザのデベロッパーツールは非表示にする
- テストデータではなく、リアルに見えるサンプルデータを使用する
- Shopify管理画面の埋め込み状態で撮影する

---

## スクリーンキャスト動画の構成案

**長さ**: 30〜60秒

1. **(5秒)** ダッシュボードを表示、「新規作成」をクリック
2. **(10秒)** ステップ1: 事業者情報を入力
3. **(5秒)** ステップ2: テンプレートを選択
4. **(5秒)** ステップ3: プレビューを確認
5. **(5秒)** 「ページを作成する」をクリック
6. **(5秒)** 成功バナー → ダッシュボードに戻る
7. **(5秒)** ダッシュボードで作成されたページを確認

## DevConsoleエラーチェックリスト（T0-7）

アプリ申請前に以下のフローを手動で確認し、Console/Networkエラーがゼロであることを確認する。

### テストフロー

- [ ] アプリをインストール（開発ストア）
- [ ] ダッシュボード画面が正常に表示される
- [ ] 「新規作成」→ ステップ1（情報入力）が正常に表示される
- [ ] ステップ1 → ステップ2（テンプレート選択）への遷移が正常
- [ ] ステップ2 → ステップ3（プレビュー）への遷移が正常
- [ ] 「ページを作成する」クリック → 成功バナー表示
- [ ] ダッシュボードに戻り、作成されたページが一覧に表示される
- [ ] 既存ページの「編集」→ ウィザードで編集 → 「ページを更新する」が正常動作

### Consoleチェック項目

- [ ] JavaScript エラー（赤色）がゼロ
- [ ] 未キャッチの Promise rejection がゼロ
- [ ] SameSite cookie 警告がゼロ（またはShopify由来のみ）
- [ ] CORS エラーがゼロ

### Networkチェック項目

- [ ] 4xx / 5xx レスポンスがゼロ（正常フロー内）
- [ ] API リクエストがすべて HTTPS
- [ ] mixed content 警告がゼロ
