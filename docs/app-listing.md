## 詳細説明（500文字以内）

「まるっと法務作成」は、日本のEC事業者がストアに必要な法的ページをまとめて作成できるShopifyアプリです。

ウィザード形式の3ステップで、必要な情報を入力するだけ。テンプレートを選んでプレビューを確認したら、ワンクリックでShopifyストアにページを作成できます。

対応ページ：
- 特定商取引法に基づく表記（無料）
- プライバシーポリシー（Basicプラン）
- 利用規約（Basicプラン）
- 返品・交換ポリシー（Basicプラン）
- 最終確認画面（Basicプラン / Theme App Extension）

2022年改正特商法（第12条の6）に対応した最終確認画面機能も搭載。カートページに法定6項目を表示できます。

面倒な法務ページの作成を、もっとシンプルに。

## 機能リスト

1. 特定商取引法に基づく表記ページをウィザード形式で簡単作成
2. プライバシーポリシーページの自動生成
3. 利用規約ページの自動生成
4. 返品・交換ポリシーページの自動生成
5. 2022年改正特商法対応の最終確認画面（Theme App Extension）
6. 法令準拠テンプレートで入力漏れ・記載ミスを防止
7. リアルタイムHTMLプレビューで公開前に仕上がりを確認
8. ダッシュボードでページの公開状態・最終更新・テンプレート更新を一覧管理

## スクリーンキャスト動画の構成案

**長さ**: 45〜90秒

1. **(5秒)** ダッシュボードを表示、「特商法ページを作成する」をクリック
2. **(10秒)** ステップ1: 事業者情報を入力
3. **(5秒)** ステップ2: 販売条件を入力
4. **(5秒)** ステップ3: プレビューを確認 → 「ページを作成する」をクリック
5. **(5秒)** 成功バナー → ダッシュボードに戻る
6. **(5秒)** 他のページタイプ（プライバシーポリシー等）の作成を簡単に見せる
7. **(10秒)** 最終確認画面設定ページで法定6項目を設定
8. **(5秒)** ダッシュボードで全ページの一覧を確認

## DevConsoleエラーチェックリスト（T0-7）

アプリ申請前に以下のフローを手動で確認し、Console/Networkエラーがゼロであることを確認する。

### テストフロー

- [ ] アプリをインストール（開発ストア）
- [ ] ダッシュボード画面が正常に表示される
- [ ] **特商法**: 「新規作成」→ ステップ1 → ステップ2 → ステップ3 → 「ページを作成する」→ 成功
- [ ] **プライバシーポリシー**: ウィザード3ステップ → 作成成功
- [ ] **利用規約**: ウィザード3ステップ → 作成成功
- [ ] **返品ポリシー**: ウィザード3ステップ → 作成成功
- [ ] **最終確認画面**: 設定ページで法定6項目を入力 → 保存成功
- [ ] ダッシュボードに戻り、作成されたページが一覧に表示される
- [ ] 既存ページの「編集」→ ウィザードで編集 → 「ページを更新する」が正常動作
- [ ] テンプレート更新通知がある場合、「テンプレートを更新」が正常動作

### Consoleチェック項目

- [ ] JavaScript エラー（赤色）がゼロ
- [ ] 未キャッチの Promise rejection がゼロ
- [ ] SameSite cookie 警告がゼロ（またはShopify由来のみ）
- [ ] CORS エラーがゼロ

### Networkチェック項目

- [ ] 4xx / 5xx レスポンスがゼロ（正常フロー内）
- [ ] API リクエストがすべて HTTPS
- [ ] mixed content 警告がゼロ
